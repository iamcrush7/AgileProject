- Select "agile" database
- Click SQL tab

## 🛠️ How It Works

### Login Flow:
1. User submits login form at `login.php`
2. JavaScript sends AJAX request to `login_handler.php`
3. Backend validates email and password
4. Password is verified using `password_verify()`
5. If valid, session is created
6. User redirected to dashboard

### Registration Flow:
1. User fills form at `register.php`
2. JavaScript sends AJAX request to `register_handler.php`
3. Email is checked for duplicates
4. Password is hashed using `password_hash()`
5. User data saved to database
6. Redirected to login page

## 📊 Database Tables

### users
- Stores user accounts
- Password hashed (never plain text)
- Types: customer, provider, admin

### services
- Available services
- Default services included:
  - Electrician
  - Plumber
  - Cleaning
  - Carpenter
  - Painter
  - Mechanic

### provider_services
- Links providers to services
- Stores pricing and ratings

### bookings & reviews
- Manages service bookings
- Stores customer reviews

## 📝 Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Connection failed | Check MySQL is running |
| User not found after register | Run database_schema.sql |
| Admin login fails | Use exact credentials above |
